# Changelog

## v1.0.0 (2026-07-05)

- Full interactive network learning platform with topology simulator and pro analysis module
- 14 device types, 8 preset scenarios, animated packet flows
- OSI 7-layer reference, TCP handshake walkthrough, protocol header visualizer
- Attack lab: ARP spoofing, SYN flood, VLAN hopping, DNS amplification
- Protocol walkthroughs: DHCP DORA, DNS resolution, TLS 1.3, OSPF neighbor
- Live network tools: ping, traceroute, subnet calculator, nslookup
- Real-time graphs, NAT table, ARP cache, quiz module
- Modular source architecture (`js/core/`) with Rollup bundling
- Unit test suite (vitest) with CI pipeline (GitHub Actions)
- CodeQL security scanning, dependency auditing
- GitHub Pages deployment automation
- Sidebar resizing, keyboard shortcuts (Space=f, c=connect, h=right panel)
- Editable link properties (type, bandwidth, latency, duplex)
- Simulation profiles (normal, congested, lossy, intercept)
- MIT licensed
